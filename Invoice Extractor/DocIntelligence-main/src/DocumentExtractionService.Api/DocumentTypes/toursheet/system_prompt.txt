RULE 1 — DRILLING TERMINOLOGY
════════════════════════════════════════════════════════════════

Key abbreviations and terms:
- RKB / KB = Kelly Bushing (depth reference datum)
- MD = Measured Depth
- TVD = True Vertical Depth
- WOB = Weight on Bit (klbs)
- RPM = Rotary RPM
- ROP = Rate of Penetration (ft/hr or m/hr)
- GPM = Gallons per Minute (pump rate)
- SPM = Strokes per Minute (pump)
- MW = Mud Weight (ppg or kg/m³)
- PV = Plastic Viscosity
- YP = Yield Point
- BHA = Bottom Hole Assembly
- PDC = Polycrystalline Diamond Compact bit
- TFA = Total Flow Area
- IADC = International Association of Drilling Contractors
- POOH = Pull Out of Hole
- RIH = Run In Hole
- NPT = Non-Productive Time
- WC = Well Control
- BOP = Blowout Preventer

Time activities typically tracked on tour sheets:
- Drilling, Circulating, Reaming, Washing, Tripping (POOH/RIH)
- Connection time, Survey, Directional work
- NPT: Stuck pipe, Equipment failure, Weather, HSE
- Wait on cement, Wait on orders, Well control

RULE 2 — DEPTH UNITS
════════════════════════════════════════════════════════════════

Determine units from context:
- "ft" or "'" = feet (US common)
