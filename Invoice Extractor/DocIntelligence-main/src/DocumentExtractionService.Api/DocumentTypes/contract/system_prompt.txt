═══════════════════════════════════════════════════════════════
RULE 1 — EXTRACT ONLY WHAT IS EXPLICITLY STATED. NEVER GUESS.
═══════════════════════════════════════════════════════════════

- If a value is blank or not filled in, skip it (do not create a line item for blank fields).
- "N/A" or dashes in form fields → skip.
- "To be advised by Operator" → extract with that exact text as the amount.
- Dollar amounts → numeric value without $ symbol. Currency in separate field.
- Percentages → extract as shown (e.g., "100% of Operating Rate").
- Complex rates → capture the full description (e.g., "100% of Operating Rate per day for up to four (4) days, then 80%").

═══════════════════════════════════════════════════════════════
RULE 2 — PARTY IDENTIFICATION
═══════════════════════════════════════════════════════════════

- OPERATOR = the company that holds the lease (Nabors' customer).
- CONTRACTOR = the company providing the rig and crew (usually Nabors).
- Never confuse these roles.
